Друге практичне завдання. Варіант 4. Виконав Губін Ярослав ТВ-23.
Спочатку створюємо рандомні масиви, де користувач задає межі чисел в масиві, кількість елементів масиву та кількість самих масивів.
Генеруємо масив за заданими параметрами.
Далі кожен масив ділиться на підмасиви по 10 елементів, для прикладу масив з 50 елементів, розіб'ється на 5 підмасивів по 10 елеменітв. 
Далі кожен з цих підмасивів заноситься в пул для обробки.
Коли всі масиви були поділені і кожен підмасив був занесений у пул для обробки, починається обробка цих масивів пулом потоків. 
І надалі користувачу виводиться результат з квадратами масивів.
А також  виводиться час, за який програма була виконана (час на ведення даних користувачем не рахується).
